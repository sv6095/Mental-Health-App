# InnerEcho: Mental Health Tracker App 📱

InnerEcho is an innovative mental health tracking application designed to help users monitor and improve their mental well-being. It offers a variety of features to track mood, set mental health goals, and access resources for better mental health management.

---

## 🌟 Features
- **Mood Tracking**: Log daily moods and visualize trends over time.
- **Goal Setting**: Set and achieve mental health goals.
- **Resource Library**: Access articles, videos, and tools on mental health.
- **Personalized Insights**: Receive tailored recommendations based on your mental health patterns.
- **Cross-Platform Support**: Available on Android and iOS devices.

---

## 🛠️ Technologies Used
### **Frontend**
- **Dart**: Flutter framework for cross-platform development.

## 🚀 Getting Started

### Prerequisites
- Flutter installed in your development environment.
- Dart SDK installed.
- Xcode (for iOS development) or Android Studio (for Android development).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sv6095/Mental-Health-App.git
   cd Mental-Health-App
   ```

2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

3. Run the Flutter application:
   ```bash
   flutter run
   ```
